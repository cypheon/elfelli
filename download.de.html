Entwicklungs-Versionen

Elfelli nutzt Git zur Quelltextverwaltung. Um die neuste Entwicklungs-Version aus dem Repository herunterzuladen, führen Sie die folgende Anweisung in einer Shell aus:


git clone https://git.sinyax.net/cypheon/elfelli.git


Sie können auch den aktuellen Entwicklungs-Snapshot herunterladen.
