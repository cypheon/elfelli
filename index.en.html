Introduction

Elfelli is a tool to simulate and visualize electric flux lines around arbitrary positioned, electric charged bodies. It is able to export PNG files of the current canvas. Elfelli is written in C++ and uses gtkmm as interface for the popular GTK+ GUI toolkit.

In current state of development, it supports point-shaped and plate-shaped bodies of either positive or negative charge. Scenes can be saved and loaded and it's possible to change the charge of any body.

The Elfelli project was kindly hosted by Gna! (RIP) from 2006 - 2017.
